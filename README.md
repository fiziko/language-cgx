# Atom language support for cgx

Syntax highlighting for Calculix: fbd, fbl

## Preview
![Preview][preview]

## Installation

```sh
apm install language-cgx
```
or install from Atom "Settings View".

## Usage

Open file *.fbd or *.fbl.
